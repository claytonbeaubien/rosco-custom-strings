# Rosco Custom Strings

A browser-based custom guitar string pack generator for Rosco Guitars Ltd.

Built for internal use and eventually customer-facing ordering via RoscoGuitars.com.

## Features
- 6, 7, 8 string guitar and 4, 5 string bass configurations
- Scale length and tuning selection
- Per-string gauge and note adjustment
- Balanced tension re-balancing
- Printable custom pack label output (Canvas-based)

## Stack
- Pure HTML / JavaScript / Canvas
- `rosco_string_engine.json` — string data source
